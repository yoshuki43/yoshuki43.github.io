Octpassインストール方法

Feb 9th, 2018 2:13 pm

Octpassとは

OctopressはJekyllを使ったブログ構築用フレームワーク。 GitHub Pagesと連携できる。

Octpass
http://octopress.org/

インストール
http://octopress.org/docs/setup/

テーマ一覧
https://github.com/imathis/octopress/wiki/3rd-Party-Octopress-Themes

インストール

準備

ruby環境の構築

sudo apt-get update\ sudo apt-get install -y ruby ruby-dev git\ git clone https://github.com/rbenv/rbenv.git ~/.rbenv\ cd ~/.rbenv && src/configure && make -C src\ git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build

以下の行を\~/.bash_profileに追記

export PATH="$HOME/.rbenv/bin:$PATH"\ eval "$(rbenv init -)"

反映

source ~/.bash_profile

ruby環境を作成(バージョンはここのrubyと同じにするとよい。)

rbenv install 2.4.2\ rbenv global 2.4.2\ gem install bundler\ #rakeは特定バージョンが必要なのでインストールしないこと！rake

node.jsをインストール(ExecJSで必要になるため。)

curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -\ sudo apt-get install -y nodejs

Octpassインストール

git clone git://github.com/imathis/octopress.git octopress\ cd octopress\ rbenv rehash\ bundle install\ rake install

テーマ変更

このへんからよさげなテーマを選ぶ。

https://github.com/imathis/octopress/wiki/3rd-Party-Octopress-Themes

GitHubをDeploy先として登録

「ユーザ名.github.io」というリポジトリをGitHubに作っておく。

$ rake setup_github_pages\    :\ Repository url: https://github.com//.github.io\    :\ rake generate\ rake deploy

「https://.github.io/」にアクセスしてみる。

設定

vi _config.yml

ブログ以外のファイルも入れたい?
-> sourceフォルダにGitHub Pagesにアップロードしたいファイルを入れる。

ブログを書く

ファイルが作成されるので、編集する。

$ rake new_post['1st post']\ mkdir -p source/_posts\ Creating new post: source/_posts/2018-02-09-1st-post.markdown

マークダウンヘルプ
https://github.com/higuma/markdown_cheat_sheet

VSCodeを使用していれば、Ctrl+Shift+Vでプレビュー画面を表示できる。

かけたら先ほどと同じようにgenerate->deployする。

rake generate\ rake deploy
